## 一、核心概念

### 1. SNR 水平（水平主力）
- SNR 核心：把「看不見的主力」畫成可觀察的**水平線（支撐 / 壓力）**。
- 水平不是預測的，是價格行為「產生」或「被我們定義」出來，再用後續走勢驗證。

### 2. 經典水平（Type 1 / 經典水平）
- 定義：**一根綠 K（漲）＋ 一根紅 K（跌）** 產生的轉折點 = 經典水平。
- 反向也成立：紅 K + 綠 K 同樣構成。
- 性質：**效率最高**，最值得信賴（一開始就證明有主力發生）。

![經典水平vsType2](/root/Crypto-Backtesting-Lab/docs/strategies/fig/rick1_classic.png)

### 3. Type 2 水平（設置 2）
- 定義：在**連續同色 K 棒**中間，由我們**自己定義**的一條水平。
- 本質：未必真有支撐/壓力，是我們給的；需後續驗證。
- 性質：**效率比 Type 1 弱**。但大時區（4H、日線）K 棒少，沒有 Type 2 就無法畫線，所以必要。

### 4. 水平失效規則
- 一條水平只能被**實體突破一次**。實體第二次穿過 = 失效，捨棄不用。

### 5. 故事線（多時區趨勢延續）
- 大時區確認趨勢模型成立 → 切小時區確認同模型被延續 → 才進場。
- 常用：日線→4H；4H→1H（最常用）。
- 不推薦一路切到 15 分（看 4 個時區）：任一時區違背就放棄，且小時區易受干擾。

![故事線延續](/root/Crypto-Backtesting-Lab/docs/strategies/fig/rick1_storyline.png)

## 二、進場條件

### A. 基本型（單時區）
1. 設水平（經典水平或 Type 2）。
2. 實體突破該水平。
3. 價格回採到水平區。
4. 被拒絕（觸及後彈回，產生影線）。
5. **收線後**才進場（沒看到拒絕就進 = 勝率低的單）。

### B. 故事線延續型（大→小）
1. 大時區（4H）模型成立 → 框起時間區間。
2. 切 1H，確認同方向模型發生在大時區成立**之後**（時間順序重要）。
3. 大時區模型**一定要先回採**（標記時間點）。
4. 故事線延續時**不需等回採**——小時區實體突破即確認延續。
5. 突破收線後在水平上找進場位。
6. 小時區沒延續、反向 → 果斷放棄。

---

## 三、出場 / 防守

- **停損**：以收線確認的拒絕處為基準，自訂風險%（1.5% / 2% / 3%）。
- **停利**：常見 1:1、1:1.5、1:2（右側波段，不抄底不逃頂）。
- **水平失效**：實體第二次穿過即停開新單。
- **心態**：有耐心，不為做單而做單。

---

## 四、實戰提醒

1. **真的看到拒絕才進**：沒拒絕就進 = 貴的、低勝率單。
2. **Type 2 必要但效率低**：大時區沒它沒機會；但心裡知道 Type 1 更強。
3. **故事線代價**：穩健但機會少，只能右側抓波段，不能抄底逃頂。
4. **時間順序關鍵**：小時區模型必須在大時區之後，延續才成立。
5. **大時區必須先回採**：沒回採就無法標記，延續判斷失準。
6. **多畫多驗證**：每天畫線、觀察價格是否照走，累積盤感。建立自己的「偏見」。
7. **畫線主觀性**：4H 可抓連續兩接近高點畫；極端瞬殺點不畫。每人鬆緊不同，有幫助即可。
8. **策略多樣性**：SNR 只是其一，另有更簡易策略可先入門。
