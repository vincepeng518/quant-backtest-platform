## 一、核心概念

1. **SNR 第一步：K 線圖 → 線性圖（line chart）**
   - 去影線，只看實體開收，減少雜訊。影線只是「摸過」，不代表真實支撐/壓力。
   - 畫任何線（趨勢線、S&R）都先去影線，養成習慣。

![K線轉線性圖](/root/Crypto-Backtesting-Lab/docs/strategies/fig/rick4_linechart.png)

2. **所有轉折 = 一紅一綠（或一綠一紅）K 棒**
   - 往上轉折 = 綠+紅；往下 = 紅+綠。每個轉折點 = 一個 SNR 水平（主力）。

3. **主力（水平）本質**
   - 假設轉折處有「看不見的主力」（供需失衡），發生當下無法驗證。
   - 靠後續**實體突破 + 回採 + 拒絕**三步驗證，確認有效才進場。

4. **兩種進場方式**
   - A（推薦）：實體突破 → 反向回採被拒絕 → 收線進場。
   - B（不愛）：同一水平 2-3 次測試即進。講者認為是「假設驗證假設」。

---

## 二、進場條件（多單為例，空單對稱）

1. **產生水平**：線性圖上一綠一紅轉折 → 畫 SNR 水平線。選明顯、歷史曾反應的。
2. **實體突破**：價格被**實體** K 穿過水平才算。影線摸過不算。
3. **回採 + 被拒絕**：價格回到水平線上被彈回（產生影線）。實體須貼近水平，影線太遠則放棄。
4. **收線確認 + 三要件**（缺一不可）：
   - ① 有水平/主力
   - ② 收線確認真的被拒絕
   - ③ 進場根（K 棒）實體守住水平（站上）
   - 滿足後在**收線位置**進場。

![進場三要件](/root/Crypto-Backtesting-Lab/docs/strategies/fig/rick4_entry.png)

## 三、出場 / 防守

1. **止盈最簡單 1:2**：止盈距 = 止損距 × 2。新手無腦用。
2. **止損經驗值**：1H 時區抓 1.5%-2%（近期保守 2%）。新手 1.1%-1.2% 也可。ATR 可自動放寬。
3. **TP 分段**：先 1:2 平一部分 → 再 1:2 平一部分 → 剩餘讓跑。
4. **套保時機**：開單已定風險，不應套保（負 EV）。例外：突發事件。更好用固定金額止損（每單損 50 元）。
5. **提早出場**：看錯方向該損就損，利潤要抱。

---

## 四、實戰提醒

1. **上下雙拒絕（區間）易失敗**：上不去也下不來，進場可能 fail，只剩經驗判斷。
2. **紀律重於一切**：嚴格執行看到的信號，不偷跑不多等。
3. **時區建議**：不熟只看 1H；厲害才往小看。頂多 15 分確認。
4. **一定要看以太定方向**：進場前看以太多空，以太往下你再漂亮也死。
5. **技術分析是機率**：完美進場也可能損。SNR+趨勢線組合勝率更高，但不該當唯一標準。
6. **斐波活用**：不懂斐波也可抓反彈（進場點 0.75 或日線高點當 TP）。
