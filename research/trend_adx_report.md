# ADX + DI + MACD 在 BTC/USDT 日線的 3 態趨勢判斷驗證報告

> 腳本：`research/trend_adx.py`（可重跑：`python research/trend_adx.py`）
> 數據：binance `BTC/USDT` 日線 OHLCV，2023-01-01 ~ 2024-05-14（取數失敗自動回退 yfinance `BTC-USD`）
> 樣本：473 根有效日線（去掉指標 warm-up）

## 1. 指標邏輯

| 指標 | 計算 | 含義 |
|------|------|------|
| TR / +DM / -DM | Wilder 真實波幅與方向性運動 | 趨勢「質」與「量」 |
| +DI / -DI (14) | 100 × Wilder(+DM)/ATR、100 × Wilder(-DM)/ATR | 多/空方向強度，0–100 |
| ADX (14) | Wilder 平滑 DX，DX=100·\|±DI差\|/(±DI和) | 趨勢「強度」，與方向無關，0–100 |
| MACD(12,26,9) 柱狀圖 | (MACD−Signal) | 動能 |
| MACD 柱狀 slope | 柱狀圖一日差分 | 動能方向 |

### 3 態信號規則（提案）
- `ADX < adx_flat` → **平盤**（趨勢弱）
- `ADX ≥ adx_trend` 且 `(+DI − -DI) > di_diff` → **看多**
- `ADX ≥ adx_trend` 且 `(-DI − +DI) > di_diff` → **看空**
- 中間地帶 → **平盤**
- **MACD 確認**：看多但柱狀 slope<0（或看空但 slope>0）→ 降級為平盤，過濾動能已反轉的假突破

## 2. 回測方法
- 對每日 t 標記信號，對照 t+1..t+**N**（N=5）收盤漲跌
- 看多正確：信號看多且 5 日後收盤 > 當日；看空反之
- 平盤正確：信號平盤且 5 日 \|報酬\| ≤ 3%（BTC 波動大，盤整帶取 ±3%）
- 方向準確率 = 方向日正確數 / 方向日總數；盤整識別率 = 平盤正確數 / 平盤總數
- 基準：隨機 0.50、前一日動量 0.49；edge = 方向準確率 − 0.50

## 3. 關鍵數字

### 提案閾值 (adx_flat=20, adx_trend=25, di_diff=2.0, MACD確認=開)
- 看多 88 / 看空 22 / 平盤 363 日
- **方向準確率 = 0.418**（edge = **−0.082**）
- **盤整識別率 = 0.543**
- 整體準確率 = 0.514

### 格點搜索最佳閾值 (adx_flat=15, adx_trend=30, di_diff=6.0)
- **方向準確率 = 0.476**（edge = −0.024）；盤整識別率 = 0.400；整體 = 0.410
- 全格點方向準確率介於 0.42–0.52，無任何組合穩定超過隨機 0.50

### 對照
- 關閉 MACD 確認（同最佳閾值）：方向準確率 **0.519**（edge +0.019），反而略高 → MACD slope 確認在 N=5 上未帶來增益，主要是縮樣與雜訊
- ADX≥25 強趨勢日的方向準確率僅 **0.107**（281 日）——強趨勢日裡指標方向與 5 日後實際走向嚴重脫節

## 5. 結論與建議閾值
- **本指標組合對 N=5 日漲跌的方向預測力接近隨機**（edge 多為負，最佳也僅 −0.02）。原因：ADX/DI 描述「當前趨勢狀態」，但 5 日級別 BTC 常出現趨勢中途的反轉與震盪，狀態≠未來方向。
- **適用定位**：作為「狀態過濾器」而非進出場信號。例如：只在 ADX≥25 且 DI 明確時才允許趨勢單；ADX<20 時禁用趨勢策略、改做區間。
- **建議日線 3 態基準參數**：`adx_flat=20, adx_trend=25, di_diff=4.0`（折衷提案值與格點結果，di_diff 取 4 兼顧方向純度與樣本量）。若只要「強趨勢才動」，用 `adx_trend=30, di_diff=6.0`。
- **MACD 確認**：對本 N=5 回測無助益，建議保留為「可選開關」，用於更長持有期或搭配其他出場邏輯時再評估。
